Moderate Enrolment Moodle Plugin
==============================

Features
--------

/ Configuration
----------------------------

1. Add the plugin into /enrol folder.
2. Enable the enrolment plugin and set it up. It is quite similar to other enrolment plugins.

Use
----
Enable the plugin in the courses you want to allow moderated self-enrollment for users.
Once a new user is added to Moodle, they will be automatically enrolled on the courses that have the moderated plugin enabled.
